# Bot Package

자동매매 봇 관리 및 실행을 담당하는 패키지입니다.

## 주요 기능

### 봇 관리
- 자동매매 봇 생성, 수정, 삭제
- 봇 활성화/비활성화
- 봇 설정 및 매개변수 관리
- 봇 성과 모니터링

### 실행 엔진
- 24/7 자동매매 실행
- 다중 전략 동시 실행
- 리스크 모니터링 및 제어
- 비상 정지 및 복구

### 상태 관리
- 봇 실행 상태 추적
- 성과 지표 실시간 업데이트
- 오류 로깅 및 복구
- 알림 발송

## 패키지 구조

- **controller/**: 봇 관리 REST API
- **service/**: 봇 실행 비즈니스 로직
- **dto/**: 봇 관련 데이터 전송 객체
- **entity/**: 봇 설정 및 상태 엔티티
- **repository/**: 봇 데이터 액세스
- **engine/**: 봇 실행 엔진
- **scheduler/**: 봇 스케줄링 관리
